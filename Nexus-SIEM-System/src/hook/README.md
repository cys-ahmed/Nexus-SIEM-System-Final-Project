# Hooks (Additional)

Additional custom React hooks.

## Files

- **useAuth.ts**: Manages user authentication state, login/logout functions, and session persistence.
- **useLogs.ts**: Custom hook for fetching and managing log data from the backend API.
- **use-mobile.tsx**: Hook to detect mobile viewports (responsive design helper).
- **use-toast.ts**: Hook for triggering toast notifications.

*Note: This directory serves a similar purpose to `../hooks` and might be merged in the future.*

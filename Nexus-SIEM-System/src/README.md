# Source Code

The main source code directory for the Nexus SIEM React application.

## Structure

- **components/**: Reusable UI components.
- **pages/**: Route-level page components.
- **hooks/** & **hook/**: Custom React hooks for state management and logic.
- **lib/**: Utility functions, constants, and helper classes.
- **App.tsx**: Main application component and router setup.
- **main.tsx**: Application entry point.

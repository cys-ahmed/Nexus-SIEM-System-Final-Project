# UI Components

This directory contains reusable, atomic UI components. These are often built on top of Radix UI primitives and styled with Tailwind CSS (following the shadcn/ui pattern).

## Components

- **Button**: Standard button component with variants (default, destructive, outline, etc.).
- **Input**: Text input fields.
- **Card**: Container for grouping related content.
- **Toast**: Notification messages.
- **Table**: Data display tables.
- **Dialog/Sheet**: Modals and slide-over panels.
- **Avatar**: User profile images.
- **Badge**: Status indicators.

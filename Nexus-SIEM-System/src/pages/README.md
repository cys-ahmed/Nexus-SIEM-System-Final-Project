# Pages

Top-level components that represent specific routes in the application.

## Categories

- **Authentication**: `Login.tsx`, `ResetPassword.tsx`, `ChangePassword.tsx`.
- **Dashboards**: `AdminDashboard.tsx`, `UserDashboard.tsx`.
- **Monitoring**: `Logs.tsx`, `Alerts.tsx`, `Availability.tsx`.
- **Incidents**: `AdminIncidents.tsx`, `UserIncidents.tsx`.
- **Profile**: `UserProfile.tsx`.

These components are typically rendered by the router in `App.tsx`.
